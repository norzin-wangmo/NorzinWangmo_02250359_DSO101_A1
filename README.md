# Todo App - DSO101 Assignment

This project demonstrates Continuous Integration and Continuous Deployment using Docker, GitHub, and Render.

## Features
- Add tasks
- Delete tasks
- Display task list

## Technologies
- Node.js
- Express
- Docker
- Render
- GitHub

## Steps

1. Created backend API using Express.
2. Created simple frontend using HTML and JavaScript.
3. Dockerized backend and frontend using Dockerfiles.
4. Built Docker images.
5. Pushed images to Docker Hub.
6. Deployed services using Render.

## Problems Faced

- Docker installation issues.
- Connecting frontend with backend API.
- Setting up Render deployment.

## Solution

Issues were solved by configuring Dockerfiles correctly and setting environment variables for deployment.

## Result

The Todo application runs successfully and is deployed using Render with automatic deployment through GitHub.
